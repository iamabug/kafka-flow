一个开源的个人项目，我用它来梳理整合自己接触到的知识，同时开发一些可能有用的工具。

目前主要开发了 Kafka 的相关工具，功能包括：

  • 支持配置连接多个 Kafka 集群
  • 在浏览器中生产数据
  • 在浏览器中消费数据
  • 在浏览器中同时生产和消费数据
